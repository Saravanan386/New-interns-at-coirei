Platform Overview

What is this LMS?

A robust backend-powered Learning Management System built for instructors and students. It supports live video classrooms, batch management, automated attendance, assignment tracking, in-platform testing, and session recordings — all accessible via a secure role-based API.

Core Features

Everything You Need

All the tools instructors and students need in one platform.

🎓

Course Management

Create and manage multiple courses, each with its own batches, modules, chapters, and lesson content. Students see their progress percentage in real-time.

📹

Live Video Classes

Power live classes via the 100ms SDK. Instructors get a Host URL; students get a secure Guest URL. Sessions are tracked with start and end timestamps.

✅

Automatic Attendance

Attendance is tracked automatically. Students present for ≥ 30 minutes are marked Present; those who leave early are marked Absent.

📋

Assignments

Instructors can post assignments linked to specific courses. Students can view assignments, track due dates, and monitor submission status.

🧪

Online Tests / Quizzes

Conduct timed tests and quizzes within the platform. Instructors create tests per module; students attempt and receive instant scores.

🎬

Session Recordings

All live sessions can be recorded. Students can replay class recordings at any time for revision. Instructors control recording access per batch.

Live Video Classes

How Live Sessions Work

Powered by the 100ms platform for reliable, low-latency live video classes with automatic role-based access.

🧭 Session Lifecycle

1

Instructor Starts Session

Calls /sessions/start → 100ms room is created. Host URL and Guest URL are generated.

2

Students Join

Students call /sessions/join. The system validates enrollment and starts the attendance timer.

3

Attendance Auto-Resolved

After 30 minutes: students still in the room → Present ✅. Students who left early → Absent ❌.

4

Session Ends

Instructor calls /sessions/end. All open records are finalized; no-shows get Absent status.

Attendance System

Automated Attendance Tracking

No manual marking needed. The system tracks join time, leave time, and duration automatically.

⏱️

Auto Timer

An async background task starts the moment a student joins — no manual intervention needed.

📐

Duration Calculation

Join time and leave time are recorded in IST. Duration in minutes is calculated and stored per student per session.

🏷️

Status Labels

Each record carries one of three statuses: Pending (in progress), Present, or Absent.

🚫

No-Show Tracking

When a session ends, enrolled students who never joined are automatically marked Absent.

Student Enrollment

Enrolling Students

Instructors enroll students with a single API call. The system automatically generates a unique Student ID and a secure random password.

🆔 Auto-Generated Student IDs

Student IDs are derived from the course name. For example:

Course NameStudent ID Format
AI / ML Frontier AI EngineerAI001, AI002, AI003…
System and Software ProSS001, SS002, SS003…
Python BasicsPY001, PY002, PY003…

🔑 Auto Password: An 8-character alphanumeric password is generated and returned in the API response. Instructors share these credentials with the student.

🔄 Password Reset: Instructors can reset any student's password via POST /enroll/reset-password?student_id=AI001

Class Scheduling

Weekly Timetable

Instructors can set a weekly recurring schedule for each batch. The system auto-generates a list of the next 3 upcoming sessions, with live status indicators.

📅 Schedule Features

Set Weekly Schedule

Configure sessions for any day of the week: Monday–Sunday. Each entry includes session type (Class / Doubt Clearing), start time, and end time.

Upcoming Sessions

API returns the next 3 upcoming scheduled dates for a batch. Shows whether a session is live right now with a direct join URL.

Two Session Types

class — Regular lecture session
doubt_clearing — Q&A / doubt solving session

API Reference

All API Endpoints

All endpoints require a Bearer JWT token (except login/register). Base URL: http://<server>:8000

🔐 Authentication

POST
/auth/register
Register a new user (admin / instructor / student)
POST
/auth/login
Login and receive a JWT access token

📚 Courses

GET
/courses/my
Get all courses the logged-in user is enrolled in, with progress

🎓 Enrollment

GET
/enroll/courses
List all courses for the enrollment dropdown
GET
/enroll/batches?course_id=
List available batches for a given course
GET
/enroll/generate-id?course_id=
Preview the next auto-generated student ID
POST
/enroll/student
Enroll a student (creates user + auto ID + auto password)
POST
/enroll/reset-password
Reset a student's password (returns new password)

📹 Live Sessions

POST
/sessions/start?course_id=&batch_name=
Instructor: Start a live session, get host & guest URLs
GET
/sessions/active?course_id=&batch_name=
Check if a session is live; get role-appropriate join URL
POST
/sessions/join?course_id=&batch_name=
Student: Join the live session and start attendance timer
POST
/sessions/leave?session_id=
Student: Leave the session and finalize attendance duration
POST
/sessions/end?session_id=
Instructor: End session, finalize all attendance records
GET
/sessions/session/{session_id}
Get attendance records for a specific session

📅 Schedule

POST
/schedule/set
Instructor: Set the weekly recurring schedule for a batch
GET
/schedule/upcoming?course_id=&batch_name=
Get the next 3 upcoming scheduled sessions (with live status)
GET
/schedule/?course_id=&batch_name=
View the complete weekly timetable for a course batch

📦 Modules & Chapters

POST
/modules/
Instructor: Create a new module for a course
POST
/modules/{module_id}/chapters
Instructor: Add a chapter to an existing module
GET
/modules/?course_id=&batch_name=
Get all modules and chapters for a course (nested)

✅ Attendance

POST
/attendance/join?session_id=
Student: Mark join time for attendance
POST
/attendance/leave?session_id=
Student: Mark leave time for attendance
GET
/attendance/session/{session_id}
Instructor: View full attendance report for a session

📋 Assignments

GET
/assignments
Get all assignments for the logged-in user's courses

📊 Dashboard

/dashboard/courses
